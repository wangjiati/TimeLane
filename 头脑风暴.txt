这是一个时间轴的可视化图表。
与甘特图有相似之处，甘特图Y轴高度通常是固定的，这里Y轴方向不固定（色块高度可变）。
充分利用图表的 X,Y,颜色,文字表达在时间轴上的关系。
通过此图可以观察到一段时间内的空闲程度、繁忙程度等。


===== 概念 =====

0: X轴：时间维度

1: 通道 (Channel)
   1.1 图表上有多个通道（至少一个），多个通道上下垂直堆叠（多行布局）
   1.2 每个通道由多个色块表达
   1.3 同一通道内色块共享同一基线（Y=0）
   1.4 通道头部：2行文字（标题 + 说明）
   1.5 通道底部：1行说明文字（可选）

2: 色块 (Block)
   2.1 色块表达一个连续时间的事项（起始时间→结束时间），沿X轴方向
   2.2 色块高度（Y轴方向）表达事项的一个属性，使用百分比表示，避免超出最大高度
   2.3 默认仅支持正值：色块从基线向上绘制
   2.4 若数据含负值：色块从基线向上下两侧绘制（正值向上，负值向下）
   2.5 同一通道内的色块允许重叠，取决于原始数据
   2.6 色块为纯色填充，本版本不使用渐变

3: 色块颜色 (Block Color)
   3.1 由使用方在数据中直接填写色值（如 #FF5733）
   3.2 未填写则使用默认主题色
   3.3 工具本身不提供颜色到属性的自动映射

4: 色块文字 (Block Text)
   4.1 色块上方或中间可显示文字
   4.2 文字可配置：颜色、是否显示、大小等属性
   4.3 文字内容与色块属性关联

5: 通道文字 (Channel Text / Event Marker)
   5.1 独立于色块，关联时间轴的某个时间点
   5.2 用于事件标注（如"上菜"、"结账"等），由使用方自由发挥
   5.3 文字可配置：颜色、是否显示、大小等属性


===== 基础配置 =====

1: 画布与通道
   1.1 通道高度 (channelHeight)：色块区域像素高度
   1.2 通道间距 (channelGap)：通道之间的间隔像素
   1.3 标签区宽度 (labelWidth)：左侧通道标签列的宽度
   1.4 头部高度 (headerHeight)：通道头部 2 行文字区域高度
   1.5 底部高度 (footerHeight)：通道底部 1 行文字区域高度

2: 时间轴样式
   2.1 时间轴区域高度 (timeAxisHeight)
   2.2 时间刻度层级：大刻度 / 小刻度
   2.3 时间格式：如 HH:mm / YYYY-MM-DD
   2.4 网格线：纵向时间网格线（可选）
   2.5 当前时间线：标识"现在"的竖线（可选）

3: 文字相关
   3.1 通道文字默认颜色（事件标注）
   3.2 色块文字默认颜色
   3.3 头部标题/说明文字颜色
   3.4 字体族、默认字号

4: 色块相关
   4.1 默认主题色（色块未填色时使用）
   4.2 色块边框：圆角、颜色、粗细
   4.3 最小可见高度/宽度（像素，防止百分比过小不可见）

5: 交互相关
   5.1 缩放范围：最小/最大缩放比例
   5.2 导出分辨率：SVG/PNG 缩放倍率

===== 交互功能 =====

1: 缩放 (Zoom) — 时间轴水平缩放，查看宏观/微观
2: 平移 (Pan) — 拖拽平移时间轴
3: 色块详情 — hover/点击色块查看属性信息
4: 导出图片 — 支持 SVG、PNG 格式


===== 技术方案 =====

1: 实现方式：Web 纯前端页面，Canvas 或 SVG 渲染
2: 导出格式：SVG、PNG
3: 数据输入：JSON 格式


===== 示例场景 =====

模拟一个饭店每桌的收入情况：
- 一个通道 = 一张桌子
- 一个色块 = 一桌客人的就餐
- 色块长度（X轴） = 桌子的占用时间
- 色块高度（Y轴，百分比） = 该桌收入（相对于最高收入），从基线向上绘制（仅正值）
- 色块颜色 = 满意度，由模拟数据直接填写色值（鲜艳=满意，暗淡=不满意）
- 通道文字 = 事件标注（如"上菜"、"结账"等）
